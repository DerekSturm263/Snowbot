## Introducing [Snowbot](https://discord.com/oauth2/authorize?client_id=1051258033931563118&permissions=1101659162624&integration_type=0&scope=bot), a great way to increase your server's engagement!

Click the above link to invite Snowbot to your server for free! From there, you can enjoy its variety of features.

- [ ] Collect Snow
- [ ] Pack Snow
- [ ] Throw Snow
- [ ] Build Buildings
- [ ] Weather Features

- [ ] Social Features 
  - [ ] Leaderboard
  - [ ] Stats
- [ ] Opt In/Out

## Issues/Feedback/Proposals
Please report any issues by adding them to the [GitHub Issues](https://github.com/DerekSturm263/Snowbot/issues) list. Feedback and feature proposals can also be submitted through there. You will be notified if your feedback/proposal is accepted into the feature backlog.
